# Brand System Rules (Enforceable)

## Palette

- Allowed tokens only: `#1A3A52`, `#FFFFFF`, `#4A90E2`, `#2C3E50`, `#7F8C8D`, `#ECF0F1`.
- Gradients and opacity-based decorative overlays are disallowed.
- Accent color usage is limited to one focal element per slide surface.

## Typography

- Preferred family: `IBM Plex Sans`.
- Minimum body size for TV-oriented content: `18px`.
- Body copy line-height MUST be `>= 1.5`.
- Body paragraphs MUST be left-aligned.

## Composition

- Safe margin MUST be at least `8%` on every edge.
- Body text blocks SHOULD remain within 6-8 lines whenever possible.
- Accent bars/dividers MUST stay thin (`2px`-`3px`) and edge-anchored.
