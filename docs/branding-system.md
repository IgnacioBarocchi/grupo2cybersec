# Branding System

This guide mirrors enforceable constraints from `.cursor/rules/brand-system.md`.

## Color Logic

- Base dark: `#1A3A52`
- White: `#FFFFFF`
- Accent blue: `#4A90E2`
- Neutral support: `#2C3E50`, `#7F8C8D`, `#ECF0F1`

Allowed high-level pairings:
- Dark base + white text + accent blue
- White base + dark text + accent blue
- Gray-dark base + white text + accent blue
- Gray-light base + gray-dark text + accent blue

## Typography

- Family: IBM Plex Sans
- TV-oriented floor: body text never under 18px
- Body line-height: 1.5 or higher

## Composition Principles

- Keep one accent focal point per slide.
- Keep safe margin at 8%.
- Prioritize readability over decorative density.
